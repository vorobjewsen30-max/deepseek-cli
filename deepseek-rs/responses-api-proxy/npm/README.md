# deepseek-cli-responses-api-proxy

<p align="center"><code>npm i -g deepseek-cli-responses-api-proxy</code> to install <code>deepseek-responses-api-proxy</code></p>

This package distributes the prebuilt [DeepSeek Responses API proxy binary](https://github.com/vorobjewsen30-max/deepseek-cli) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/deepseek-responses-api-proxy.js --help
```

Refer to [`codex-rs/responses-api-proxy/README.md`](https://github.com/vorobjewsen30-max/deepseek-cli) for detailed documentation.
