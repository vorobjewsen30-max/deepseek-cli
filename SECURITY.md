# Security Policy

Thank you for helping us keep DeepSeek secure!

## Reporting Security Issues

The security is essential to DeepSeek's mission. We appreciate the work of security researchers acting in good faith to identify and responsibly report potential vulnerabilities, helping us maintain strong privacy and security standards for our users and technology.

Our security program is managed through Bugcrowd, and we ask that any validated vulnerabilities be reported via the [Bugcrowd program](https://bugcrowd.com/engagements/deepseek).

## Vulnerability Disclosure Program

Our Vulnerability Program Guidelines are defined on our [Bugcrowd program page](https://bugcrowd.com/engagements/deepseek).

## How to operate CODEX safely

For details on DeepSeek security boundaries, including sandboxing, approvals, and network controls, see [Agent approvals & security](https://platform.deepseek.com/docs).
